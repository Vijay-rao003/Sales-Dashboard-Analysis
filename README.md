# Sales-Dashboard-Analysis
Excel Dashboard 
Project is an interactive Sales & Financial Dashboard designed to analyze orders, revenue, customer spending, delivery performance, and sales trends.
The dashboard provides a clear business view of what is selling, when it is selling, and where it is selling, enabling data-driven decision-making.
The project demonstrates strong skills in data analysis, KPI tracking, dashboard design, and business storytelling.
Project Objectives:
Track overall sales performance
Monitor total orders, revenue, and delivery efficiency
Analyze revenue by occasion, category, and month
Identify top-performing cities
Enable time-based and occasion-based filtering
